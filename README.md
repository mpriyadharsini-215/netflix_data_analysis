# Netflix Data Analysis Project

## 📌 Overview
This project analyzes Netflix’s dataset using Python to understand trends in movies and TV shows, content growth over time, country-wise production, genres, and audience ratings.

## 🛠️ Tools Used
- Python  
- Pandas  
- Matplotlib  

## 📊 Key Analyses

### 📈 Content Growth Over Time
Netflix shows rapid growth between 2015–2018, followed by a slowdown in later years.

### 🌍 Country Analysis
Countries like Kuwait, Taiwan, and Romania show high recent growth, indicating Netflix expansion into emerging markets.

### 🎬 Genre Analysis
International Movies and Drama dominate Netflix, showing focus on global and emotional content.

### ⭐ Ratings Analysis
Most content falls under TV-MA and TV-14, indicating focus on mature audiences.

## 📌 Final Insight
Netflix follows a global content strategy with strong emphasis on international expansion, diverse genres, and mature audience targeting.
